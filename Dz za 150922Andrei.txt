(1) Зачем предназначен файл .gitignore?
      говорит Git  какие файлы файлы будет игнарировать при фиксации в GitBush

(2) Что делает команда git remote rm origin?
       привязка к git

(3) В каких случаях происходит MERGE CONFLICT?
    При слияни после внесеных изменений в разных ветках 
и Git не понимает какие изменения оставить (с какой ветки) 
предупреждает сообщением.

(4) Как посмотреть отличия между ветками?
           git branch  –v
(5) Какой командой удалить локальную и удаленную ветки?
              git branch -D
(6) Как сделать файл не отслеживаемым (untracked)?
Надо  в .gitignore закинуть файл, после git add не будет видеть этот файл,
а будет видеть обновленный .gitignore, который нужно
